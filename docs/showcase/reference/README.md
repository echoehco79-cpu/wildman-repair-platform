# 架构看板参考使用说明

`AI维修协同中枢_架构看板_版式参考.pptx` 只是系统架构图的版式和信息层级参考，不是野人先生当前项目的业务内容依据。

## 允许参考

- 16:9 横版大画布构图。
- 接入层、核心能力层与数据/业务底座的分层思路。
- 模块框架、分层标题、连接线和单页全景表达方式。
- 大型架构图的整体信息密度。

## 禁止复用

- BPM 流程新增、停用、删除和版本管理。
- 审批节点、抄送、自动转办、自动转签和退回。
- 通用流程编辑器和表单设计器。
- OA、APP 及其他未确认外围系统。
- AI 自动审批、任务拆解、子任务与泛任务管理。
- 参考 PPT 中的原始模块名称、文案和业务分类。

## 内容优先级

1. 系统需求说明书、详细设计、真实系统代码和数据库结构。
2. 页面视觉风格和品牌规范。
3. 本目录中的 PPT 版式参考。

最终架构图不得保留任何与野人先生设备维修协同无关的 BPM 平台能力。
